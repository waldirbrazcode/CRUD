# Project requirements:

- Node.js
- Python

# How to run the project:

### 1 - Clone the project for your machine

On your terminal type:
<code>git clone https://github.com/waldirbrazcode/CRUD.git</code>

### 2 - Run the server

- First, create an virtual enviroment. Open the terminal and type: <code>python -m venv .venv</code> and activate it by typing: <code>.venv/Scripts/activate</code>
- In the folder "backend" type in the terminal: <code>pip install -r requirements.txt</code> to install the dependencies
- On the terminal go to the folder "proj_back" inside "backend" and type on terminal: <code>py manage.py runserver</code>

### 3 - Run the frontend (website)

- Open the terminal and go to "proj-react" inside frontend
- Install the dependencies with the following commands on terminal:
  
  <code>npm install vite</code>
  
  <code>npm install tailwindcss @tailwindcss/vite</code>
  
- Type in the terminal: <code>npm run dev</code>
- Open the local server
